# CS-370-Pirate-Intelligent-Agent

Pirate Intelligent Agent Portfolio Submission

For this project, I worked on a pirate intelligent agent that used reinforcement learning to solve a pathfinding problem. The goal was for the agent to learn how to move through the environment and reach the treasure. This project helped me understand how reinforcement learning can be used to train an agent through rewards, decisions, and repeated learning.

Some of the code was already given to me, including the setup for the environment, the maze/game structure, and some of the helper code needed to run the project. The part I created myself was the code for the intelligent agent, including the deep Q-learning logic. I worked on how the agent chose actions, remembered past experiences, trained from those experiences, and improved over time.
